# Sample Monorepo Setup with JS and TS, storybook and styleguidist

## Setup

```
yarn
```

## Using Styleguidist and Storybook

### Styleguidist

```
yarn styleguidist
```

### Storybook

```
yarn storybook
```

## Generating new component skeletons

### JSX

```
yarn generate-jsx
```

### TSX

```
yarn generate-tsx
```
